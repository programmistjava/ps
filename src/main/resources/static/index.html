Welcome, People_Flow!

This application is a People_Flow microservice .

  • It does not have a front-end.
  • For dev profile Use login: user passwd: user
  • It is serving REST APIs, under the '/api' URLs.
  • CRUD for Employee http://localhost:8081/api/employees
  • Including creation with state "ADDED" f.e. Post request http://localhost:8081/api/employees
  • And endpoint to change the state f.e. patch request http://localhost:8081/api/employees/3/state/ACTIVE
  • OpenAPI documentation endpoint for those APIs is at /v3/api-docs
